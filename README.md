## Selected Work

### MiniCode-Python

Terminal AI coding assistant with tool use, memory, planning-execution loops, and MCP integration.

- Built a Python-based coding agent with 30+ tools and multi-level memory modules
- Implemented planning, execution, iteration, shell/Git workflows, and external model integration
- GitHub: https://github.com/QUSETIONS/MiniCode-Python

### NECIO System

Local pathology WSI analysis workflow built around CellSAM, OpenSlide, SQLite, and a human-in-the-loop GUI.

- Supports SVS, NDPI, and TIFF whole-slide image workflows
- Covers slide labeling, nuclei detection, feature extraction, local data management, and HTML report generation
- GitHub: https://github.com/QUSETIONS/necio-system

### OCTA Vessel Analyzer

Biomedical image analysis tool for OCTA vessel-related processing and quantitative analysis.

- Focused on medical image preprocessing, vessel analysis, and reproducible quantitative outputs
- GitHub: https://github.com/QUSETIONS/octa-vessel-analyzer

### Healthy Agent for World

Medical multi-agent diagnosis prototype connecting triage, evidence retrieval, safety checks, and report generation.

- Built around FastAPI, RAG, tool calling, confidence ranking, and audit logs
- Explores how healthcare agents can become more transparent and clinically cautious
